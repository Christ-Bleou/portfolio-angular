 npm start            Lancer le serveur
 npm run build        Construire pour production
 npm test             Exécuter les tests
 npm run cli help     Afficher l'aide CLI

🎬 PROCHAINES ÉTAPES

ÉTAPE 1: Lire
  → Vous avez 5 minutes? Lisez README.md
  → Vous avez 30 minutes? Lisez GETTING_STARTED.md

ÉTAPE 2: Installer
  → npm install

ÉTAPE 3: Configurer
  → npm run cli init

ÉTAPE 4: Voir
  → npm start
  → Visitez http://localhost:4200

ÉTAPE 5: Déployer
  → npm run build
  → Uploadez dist/ sur Vercel/Netlify/GitHub Pages
