# 📚 Portfolio Generator - Documentation

Bienvenue dans le **Portfolio Generator** ! Un outil complet pour créer votre portfolio personnel en quelques minutes.

---

## 🎯 Démarrage rapide (30 secondes)

```bash
npm install          # Installer (une seule fois)
npm run cli init     # Configurer votre portfolio
npm start            # Lancer le serveur
```

➡️ Ouvrez **http://localhost:4200** 🎉

## 🎯 Parcours selon votre besoin

### 🆕 Je suis nouveau ici

1. Lire : [GETTING_STARTED.md](./GETTING_STARTED.md#installation)
2. Exécuter : `npm install && npm run cli init && npm start`
3. Modifier : `src/assets/data/config.json`

### 🔧 Je veux configurer mon portfolio

1. Exécuter : `npm run cli init`
2. Ou éditer manuellement : `src/assets/data/config.json`
3. Consulter : [PORTFOGENERATOR_README.md](./PORTFOGENERATOR_README.md#configuration)

### 🎨 Je veux le personnaliser

1. Consulter : [GETTING_STARTED.md#personnalisation](./GETTING_STARTED.md#personnalisation)
2. Valider : `npm start`
3. Vérifier : http://localhost:4200

### 🚀 Je veux le mettre en ligne

1. Build : `npm run build`
2. Consulter : [GETTING_STARTED.md#déploiement](./GETTING_STARTED.md#déploiement)
3. Choisir une plateforme et déployer

### ❓ J'ai un problème

1. Lire : [GETTING_STARTED.md#troubleshooting](./GETTING_STARTED.md#troubleshooting)
2. Ou : [COMMANDS.md#dépannage](./COMMANDS.md#troubleshooting)
